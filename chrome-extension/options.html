Vertex AI configuration

The project where Vertex AI is enabled and quotas are filed.

Run gcloud auth print-access-token and paste the result. Token lasts ~60 minutes; refresh when you see "access token expired" in a card. Stored in chrome.storage.local, never sent anywhere except *.aiplatform.googleapis.com.

Get one free at aistudio.google.com/apikey. Leave blank to keep using Vertex for everything.
